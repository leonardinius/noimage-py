About NoImage

This is copycast and ripoff of awesome DummyImage service, developed by Russell Heimlich.

I liked the idea and nevertheless there is already python port available - Ipsum Image I decided to re-implement that myself. The decision was not driven by lack of functionality of existing solutions, the ultimate goal was (and still is) to learn and play with Python snake a little bit more. "Learning by doing" way.

Fork me on GitHub

Getting help

Found an bug or want to request a new feature? Go to NoImage issue tracker.

Have an API usage question? Try to find an answer at wiki space.

Service API

URL format: /size[/bgcolor][/txtcolor][.format][?t=text]

  • size is <width>x<height> or one of the predefined size aliases as below

  • bgcolor image background color. For detailed documentation refer to ImageColor Module documentation.

    Samples: black, #AB8CC5, hsl(0,100%,50%).

  • txtcolor foreground text color. For detailed documentation refer to ImageColor Module documentation.

    Samples: black, #AB8CC5, hsl(0,100%,50%).

  • format is one of the black, gif,jpeg,png,bmp.

  • text url-encoded. Please note: | could be used as new line character.

    Samples: ?t=text, ?t=two|line|text, ?t=%24%24%24.
