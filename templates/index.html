Service API

URL format: {{host}}/<size>[/<bgcolor>][/<txtcolor>][.<format>][?t=<text>]

  • <size> is <width>x<height> or one of the predefined size aliases as below

  • <bgcolor> [optional] image background color. Supported formats are as follows:

    • Hexadecimal color specifiers, given as "rgb" or "rrggbb". For example, ff0000 specifies pure red.
    • RGB functions, given as rgb(red, green, blue) where the colour values are integers in the range 0 to 255. For example, rgb(255,0,0).
    • Common HTML colour names, more than 140 standard colour names are supported. Colour names are case insensitive. For example, red and Red both specify pure red.

    Samples: black , AB8CC5 , hsl(0,100%,50%) .

  • <txtcolor> [optional] foreground text color. Supported formats are as follows:

    • Hexadecimal color specifiers, given as "rgb" or "rrggbb". For example, ff0000 specifies pure red.
    • RGB functions, given as rgb(red, green, blue) where the colour values are integers in the range 0 to 255. For example, rgb(255,0,0).
    • Common HTML colour names, more than 140 standard colour names are supported. Colour names are case insensitive. For example, red and Red both specify pure red.

    Samples: black , AB8CC5 , hsl(0,100%,50%) .

  • <format> [optional] is one of the gif,jpeg,png,bmp.
  • <text> [optional] url-encoded image text (NB difference: | could be used as new line character).

    Samples: ?t=text , ?t=two|line|text , ?t=%24%24%24 .
