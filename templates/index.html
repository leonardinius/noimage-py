NoImageYet

Placeholder image generator service to fill in void spaces in web-page mock-up. Created by Leonids Maslovs, inspired by DummyImage.com.

Fork me on GitHub

Getting help

Found an bug or want to request a new feature? Go to NoImageYet issue tracker.

Have an API usage question? Try to find an answer at wiki space.

Service API

URL format: /size[/bgcolor][/txtcolor][.format][?t=text]

  • size is <width>x<height> or one of the predefined size aliases as below

  • bgcolor [optional] image background color. For detailed documentation refer to ImageColor Module documentation.

    Samples: black , #AB8CC5 , hsl(0,100%,50%) .

  • txtcolor [optional] foreground text color. For detailed documentation refer to ImageColor Module documentation.

    Samples: black , #AB8CC5 , hsl(0,100%,50%) .

  • format [optional] is one of the black, gif,jpeg,png,bmp.
  • text [optional] url-encoded image text (NB difference: | could be used as new line character).

    Samples: ?t=text , ?t=two|line|text , ?t=%24%24%24 .
